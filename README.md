# RenderFever

### Dependencies
* [fmt](https://github.com/fmtlib/fmt) - An open-source formatting library providing a fast and safe alternative to C stdio and C++ iostreams.
  * Licensed under the [MIT License](https://github.com/fmtlib/fmt/blob/master/LICENSE).
  * vcpkg: `vcpkg install fmt`
* [Frozen](https://github.com/serge-sans-paille/frozen) - Header-only library that provides 0 cost initialization for immutable containers, fixed-size containers, and various algorithms.
  * Licensed under the [Apache License 2.0](https://github.com/serge-sans-paille/frozen/blob/master/LICENSE).
  * vcpkg: `vcpkg install frozen`
* [OpenGL](https://www.opengl.org/) - The Industry Standard for High Performance Graphics.
  * vcpkg: `vcpkg install opengl`
* [GLEW](https://github.com/nigels-com/glew) - The OpenGL Extension Wrangler Library.
  * Licensed under the [BSD 3-Clause License](https://github.com/nigels-com/glew/blob/master/LICENSE.txt).
  * vcpkg: `vcpkg install glew`
* [GLFW](https://github.com/glfw/glfw) - A multi-platform library for OpenGL, OpenGL ES, Vulkan, window and input.
  * Licensed under the [zlib License](https://github.com/glfw/glfw/blob/master/LICENSE.md).
  * vcpkg: `vcpkg install glfw3`
* [GLM](https://github.com/g-truc/glm) - OpenGL Mathematics.
  * Licensed under the [MIT or Happy Bunny Licenses](https://github.com/g-truc/glm/blob/master/copying.txt).
  * vcpkg: `vcpkg install glm`
* [STB](https://github.com/nothings/stb) - Single-file public domain libraries for C/C++.
  * Licensed under the [MIT License or Public Domain](https://github.com/nothings/stb/blob/master/LICENSE).
  * vcpkg: `vcpkg install stb`
* [Flecs](https://github.com/SanderMertens/flecs) - A fast entity component system (ECS) for C & C++.
  * Licensed under the [MIT License](https://github.com/SanderMertens/flecs/blob/master/LICENSE).
  * vcpkg: `vcpkg install flecs`
* [ImGui](https://github.com/ocornut/imgui) - Bloat-free Graphical User interface for C++ with minimal dependencies.
  * Licensed under the [MIT License](https://github.com/ocornut/imgui/blob/master/LICENSE.txt).
  * vcpkg: `vcpkg install imgui`
* [zlib](https://github.com/madler/zlib) - A massively spiffy yet delicately unobtrusive compression library.
  * Licensed under the [zlib License](https://github.com/madler/zlib/blob/develop/zlib.h#L4-L28).
  * vcpkg: `vcpkg install zlib`
* [assimp](https://github.com/assimp/assimp) - Open Asset Import Library.
  * Licensed under the [BSD-3-Clause License](https://github.com/assimp/assimp/blob/master/LICENSE).
  * vcpkg: `vcpkg install assimp`
